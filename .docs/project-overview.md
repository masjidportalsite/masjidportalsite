# Project Overview: Masjid Portal

A "Digital Sanctuary" for mosque administrators and congregants.

## Vision
To provide a seamless, mobile-first management platform for modern Masjids, enabling community engagement, financial transparency, and spiritual growth.

## Core Features
- **Dashboard**: Real-time community metrics.
- **Prayer Times**: Automated and manual schedule management.
- **Donations**: Secure payment processing (MYR focus).
- **Members**: Community directory and family links.
- **Events**: Program registration and volunteer shifts.
- **Branding**: Dynamic UI customization for individual mosques.

## Target Audience
- Mosque Administrators (Admins)
- Spiritual Leaders (Imams)
- Financial Officers (Treasurers)
- Community Members (Congregants)
