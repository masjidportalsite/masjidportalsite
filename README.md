# Masjid Portal Site

Masjid Portal Site is a persistent autonomous engineering project to build, maintain, and scale a SaaS platform for mosques.

## 🕌 Digital Sanctuary
We fuse the precision of modern SaaS engineering with the serenity and timelessness of spiritual architecture.

## 🛠 Tech Stack
- **Frontend**: Next.js (App Router), Tailwind CSS
- **Backend**: InsForge (PostgreSQL, Auth, Storage)
- **Deployment**: Vercel
